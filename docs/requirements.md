¿Qué es este proyecto? 
Este proyecto consiste en la creación de un CRM a medida, diseñado específicamente para vuestra operativa como asesoría energética.
No se trata de adaptar una herramienta genérica, sino de desarrollar una solución centrada en:
La gestión de clientes
El control de contratos energéticos
El seguimiento del estado de cada operación
El objetivo es disponer de una herramienta sencilla, clara y práctica que permita centralizar toda la información y mejorar la gestión diaria del negocio.
 ¿Qué incluye y qué no incluye? 
Lo que sí incluye en esta propuesta:
Panel principal (dashboard) con visión global del negocio:
Total de contratos
Estados de los contratos
Incidencias
Gestión completa de clientes:
Ficha detallada con todos los datos necesarios
Clasificación por tipo (residencial / PYME)
Gestión de contratos energéticos:
CUPS
Comercializadoras
Producto
Consumo
Precios
Tarifas
Comisiones
Seguimiento del estado de cada contrato:
Pendiente de tramitar
En tramitación
Pendiente de firma
Activo
Cancelado
Control de renovaciones
 ¿Qué vas a poder hacer con la herramienta? 
Con este CRM podrás:
Tener todos los clientes centralizados en un único sistema
Gestionar múltiples contratos por cliente
Ver en todo momento el estado de cada operación
Priorizar tareas según contratos pendientes o incidencias
Controlar renovaciones y oportunidades comerciales
Consultar de forma rápida el estado global del negocio

 ¿Cómo se valida que funciona bien?
El proyecto se considerará finalizado cuando:
Se pueda acceder al CRM y utilizarlo con normalidad
Los clientes puedan crearse, editarse y consultarse correctamente
Los contratos puedan gestionarse con todos sus datos
Los estados de los contratos funcionen correctamente
El dashboard refleje la información de forma coherente
Los datos hayan sido importados correctamente (fase 2)
Se haya realizado una validación conjunta satisfactoria
 ¿Qué pasa después de la entrega?
Una vez finalizado el proyecto y validado el CRM, la herramienta quedará completamente operativa y accesible online.
A partir de ese momento, se ofrecen opciones de mantenimiento para garantizar:
El correcto funcionamiento del sistema
Resolución de incidencias
Seguridad y disponibilidad de los datos
